Add new products to the catalog.

Steps:
1. Read @commands/data.md for product shape and existing IDs
2. Add product objects to `PRODUCTS` array in `src/store.jsx`
3. Follow ID sequence: next after p20 is p21, etc.
4. Pick hue [h,s,l] that matches the product's packaging color
5. Pick shape from: jar|bottle|tube|flask|lipstick|palette|pencil|stick|spray|bar

Product details: $ARGUMENTS
